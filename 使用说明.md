# 密安密码本使用说明

欢迎使用密安密码本。这是一款**纯本地、离线可用**的密码管理工具，你的所有密码数据都会加密保存在自己的手机或电脑上，不会上传到任何服务器。

> 本说明对应**最新版本**，如果你当前使用的文件不是最新的，请按下面“版本更新”章节替换。

---

## 一、拿到软件后怎么打开

你拿到的软件有两种形式，任选一种使用即可。最新文件都在 `dist` 目录下：

### 1. 单文件 HTML（网页版）

文件名为：`mian-password-local.html`

- **在手机上**：把文件保存到手机，用浏览器打开（推荐 Chrome、Edge、Safari 等现代浏览器）。
- **在电脑上**：双击文件，浏览器会自动打开。

> 注意：网页版无需安装，即开即用。但它不能像 App 一样安装到主屏幕，也不会自动离线缓存。

### 2. Android 安装包（App 版）

文件名为：`密安密码本.apk`

1. 把 APK 文件发送到 Android 手机。
2. 在手机上点击安装，如果系统提示“未知来源应用”，请按提示允许一次。
3. 安装完成后，打开“密安密码本”应用即可。

> 注意：APK 版所有资源都内嵌在应用里，**无需联网**也能使用。

---

## 二、首次使用：创建主密码

第一次打开软件，会出现“创建你的主密码”界面。

1. 输入一个**主密码**（建议至少 8 位，最好包含字母、数字和符号）。
2. 再次输入确认密码。
3. 可以填写“密码提示”（可选，用于提醒自己）。
4. 点击“创建保险库”。

> **非常重要**：主密码是你所有数据的唯一钥匙。如果你忘记主密码，**没有任何办法恢复**，请务必牢记或把密码存放在安全的地方。

---

## 三、主界面说明

创建主密码并解锁后，会进入主界面。界面从上到下分别是：

- **顶部标题栏**：左侧是软件名称，右侧有“密码生成器”、“设置”和“锁定”按钮。
- **搜索框**：输入平台名称、用户名或备注，可以快速查找密码。
- **密码列表**：显示已保存的密码记录。
- **右下角加号按钮**：点击可以添加新密码。

---

## 四、添加密码

点击右下角的“加号”按钮，进入添加页面：

| 项目 | 说明 |
|------|------|
| 平台 / 应用名称 | 例如：微信、支付宝、Google 等 |
| 用户名 / 邮箱 | 你的登录账号 |
| 密码 | 你的登录密码，也可以点击“生成”按钮随机生成 |
| 网址（可选） | 登录网址 |
| 分类 | 可以选择“社交、金融、工作、购物、其他”等分类 |
| 加入收藏 | 勾选后，该记录会被优先显示或便于查找 |
| 备注（可选） | 可填写安全问题、二次验证提示等 |

填写完成后，点击“保存”即可。

## 五、查看、编辑和删除密码

### 查看密码

在主界面点击某条记录，进入详情页：

- 密码默认是隐藏的（显示为圆点），点击“显示”可以查看明文。
- 点击“复制密码”可以直接复制到剪贴板，方便粘贴使用。

### 编辑密码

在详情页点击“编辑”按钮，修改后保存即可。

### 删除密码

在详情页底部点击“删除”，确认后该记录会被删除。删除操作无法撤销，请谨慎操作。

> 如果你之前用旧版本保存过密码，首页出现“未命名”且点不开详情，请看下面的“版本更新与旧数据修复”章节。

## 六、密码生成器

点击主界面右上角的“密码生成器”按钮，可打开密码生成器。

- **随机密码**：可设置长度，是否包含大小写字母、数字、特殊符号，并可排除易混淆字符（如 0 和 O）。
- **助记密码**：由多个英文单词组合，便于记忆。
- **PIN 码**：纯数字，可设置位数。

生成后点击“复制”即可使用。

---

## 七、设置与备份

点击主界面右上角的“设置”按钮，进入设置页面。

### 1. 立即锁定

点击后会退出到解锁界面，需要重新输入主密码才能进入。建议每次使用完都手动锁定。

### 2. 导出加密备份

点击“导出加密备份”，会生成一个 `.json` 文件并保存到手机或电脑。

- 备份文件包含你的所有密码数据，并且已经过加密，必须配合你的主密码才能使用。
- 建议**定期导出备份**，并保存到安全位置（如 U 盘、可信云盘等）。

### 3. 导入加密备份

如果你更换了手机或浏览器，可以点击“导入加密备份”，选择之前导出的 `.json` 文件，即可恢复所有密码。

> 注意：导入后需要使用原来的主密码解锁。

### 4. 修改主密码

点击“修改主密码”，输入当前主密码和新主密码，确认后即可修改。修改主密码后，请**重新导出一份备份**。

### 5. 重置保险库

点击“重置保险库”会清空所有密码记录和主密码，相当于恢复出厂设置。**此操作不可恢复，请谨慎使用**。

## 八、重要提醒

1. **主密码无法找回**：软件没有后台服务器，也没有“忘记密码”功能。请务必备份主密码或设置容易想起的密码提示。
2. **数据保存在本地**：所有密码只保存在你当前设备的浏览器或 App 中，卸载 App、清除浏览器数据或更换设备都会导致数据丢失。
3. **定期备份**：建议每次添加或修改重要密码后，都导出一份备份文件。
4. **备份文件要保密**：虽然备份文件已加密，但仍建议不要上传到不信任的公共平台。

## 九、版本更新与旧数据修复

### 如何更新到最新版

- **HTML 网页版**：直接用最新版的 `mian-password-local.html` 替换旧文件，用浏览器打开即可。
- **APK 版**：安装最新版 APK。如果系统提示“签名冲突”或“无法安装”，请先卸载旧版，再安装新版。
- **PWA 网页版**：如果之前通过浏览器把网页安装到了主屏幕，打开一次后会自动更新到最新版本。

### 旧数据自动修复

如果你之前用旧版本保存过密码，可能会遇到首页记录显示“未命名”、点不开详情、搜索不到的情况。这是旧版本的加密字段问题，已经修复。使用最新版文件并重新解锁后，系统会自动把旧数据转换成正确格式，原来的记录就能正常显示和编辑了。

> 建议修复后立即进入“设置”导出一份新的加密备份，以防万一。

## 十、常见问题

**Q：忘记了主密码怎么办？**  
A：无法恢复。只能通过旧设备或旧备份重新导入数据。请平时做好主密码和备份的记录。

**Q：换手机后数据还能迁移吗？**  
A：可以。在原设备上导出加密备份文件，传到新设备后使用“导入加密备份”功能即可。

**Q：同一个账号能在多台设备同步吗？**  
A：不能自动同步。数据是本地独立的，需要手动导出/导入备份来迁移。

**Q：APK 安装时提示有风险？**  
A：这是正常现象，因为软件没有上架应用商店。如果你信任该软件来源，允许安装即可。

**Q：网页版可以离线使用吗？**  
A：单文件 HTML 版打开后即可使用，但关闭浏览器后不会自动离线缓存。如需安装到主屏幕，请使用 APK 版。

**Q：首页显示“未命名”，点击详情没有反应？**  
A：这是旧版本的已知问题。请使用最新版文件，重新解锁后系统会自动修复旧数据。具体见“版本更新与旧数据修复”。
